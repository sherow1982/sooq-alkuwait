## 🎥 التأثيرات المتقدمة

### 1. تأثير النجوم (Stars Animation)
```css
background-image: radial-gradient(2px 2px at 20% 30%, white, transparent);
animation: stars 60s linear infinite;
```

### 2. توهج النيون (Neon Glow)
```css
text-shadow: 
    0 0 10px var(--neon-green),
    0 0 20px var(--neon-green),
    0 0 30px var(--neon-green),
    0 0 40px var(--kuwait-green);
```

### 3. تأثير Pulse
```css
animation: pulse 2s ease-in-out infinite;
```

### 4. تأثير التوسع الدائري
```css
.button::before {
    background: radial-gradient(circle, color 0%, transparent 70%);
    transition: width 0.6s, height 0.6s;
}
```

### 5. تأثير الدوران المضيء
```css
background: conic-gradient(from 0deg, transparent, var(--neon-green));
animation: rotate 4s linear infinite;
```

### 6. تأثير Flicker (وميض)
```css
animation: flicker 0.5s ease-in-out infinite;
```
