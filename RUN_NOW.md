# 🚀 تشغيل فوري لمولد المنتجات

## ⚡ التشغيل الفوري والشامل

```bash
node cleanup_and_run.js
```

هذا الأمر سيقوم ب:

### 🧹 تنظيف تلقائي
- حذف جميع الملفات المكررة
- إزالة الملفات غير الضرورية
- تنظيم هيكل المشروع

### 🚀 توليد تلقائي
- توليد 1,977 صفحة منتج
- إنشاء sitemap.xml
- إنشاء product-feed.xml
- تشفير آمن للروابط العربية

### ⏱️ الوقت المتوقع
- التنظيف: 5-10 ثوان
- التوليد: 30-60 ثانية
- **الإجمالي: أقل من دقيقتين**

### 🎯 النتيجة المضمونة
- 📁 مجلد `products-pages/` مملوء بالصفحات
- 🗺️ `sitemap.xml` محدث بجميع الروابط
- 📡 `product-feed.xml` جاهز لـ Google Merchant
- 🌐 موقع كامل جاهز للنشر
