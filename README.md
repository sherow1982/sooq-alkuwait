## 🚀 بدء التشغيل

### 1. توليد جميع صفحات المنتجات

```bash
# توليد 374 صفحة منتج دفعة واحدة
node generate-all-product-pages.js

# أو باستخدام npm
npm run generate-pages
```

### 2. تشغيل المتجر محلياً

```bash
# مع Python
python -m http.server 8000

# أو باستخدام npm (يولد الصفحات ويشغل السيرفر)
npm start
```

### 3. الوصول إلى الموقع

- **محلياً**: http://localhost:8000

## 🛠️ تعليمات الصيانة

### تحديث بيانات المنتجات
1. عدل ملف `products_data.json`
2. شغّل `node generate-all-product-pages.js`
3. سيتم تحديث جميع الصفحات تلقائياً

### تغيير معلومات الاتصال
- عدّل أرقام الواتساب في جميع الملفات
- غيّر الإيميل في الفوتر وأزرار الاتصال

### إضافة فئات جديدة
- عدّل دالة `categorizeProduct()` في products-catalog.html
- أضف فلتر جديد في `.filter-tabs`
