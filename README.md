# 🛒 سوق الكويت - مولد صفحات المنتجات

مولد تلقائي لصفحات المنتجات باللغة العربية مع تحسين محركات البحث (SEO)

## 🚀 التشغيل الفوري

```bash
# التشغيل السريع
node quick_run.js

# أو باستخدام npm
npm run generate

# أو التشغيل المباشر
node generate_products.js
```

## ✨ المميزات

- 🚀 توليد تلقائي لـ 1,977 صفحة منتج
- 🔤 تشفير آمن للروابط العربية
- 🗺️ إنتاج خريطة موقع تلقائية
- 📡 توليد فيد للمنتجات (Google Merchant)
- 📱 تصميم متجاوب
- 🎨 واجهة مستخدم أنيقة
- 🛒 نظام سلة تسوق

## 📊 الإحصائيات

- **المنتجات:** 1,977 منتج
- **الصفحات المولدة:** تلقائياً
- **اللغة:** العربية (RTL)
- **المنصة:** الكويت
- **الوقت:** ~2-3 دقائق

## 🎯 النتائج

بعد التشغيل ستحصل على:
- 📁 `products-pages/` - جميع صفحات المنتجات
- 🗺️ `sitemap.xml` - خريطة الموقع
- 📡 `product-feed.xml` - فيد المنتجات
